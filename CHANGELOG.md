# Changelog

## [1.0.1] - 2026-04-02

### Fixed
- 修复嵌套控件检测：父控件和子控件（如标签页和关闭按钮）现在会分别显示独立标签
- 排除禁用控件：不再标记灰色的不可点击按钮
- 排除纯文本元素：过滤 QQ 的消息数量、等级、头衔等 Text 类型控件

### Improved
- 优化控件类型检测逻辑，减少重复 API 调用

## [1.0.0] - 2026-04-01

### Added
- 初始版本发布
- 基于 UI Automation 和 Win32 API 的控件检测
- 双字符标签系统（AA-ZZ）
- 多种鼠标模式切换（左键/右键/中键/双击）
- 可配置的视觉样式（字体大小、透明度、圆角、标签位置）
