# Nexus 1.0.0 - 让桌面操作像 Vim 一样高效

Nexus 是一款 Windows 下的键盘模拟鼠标点击工具。适合不习惯频繁移动鼠标、希望用键盘完成大部分点击操作的用户。其应用程序体积小于100KB，常驻内存占用仅有2~3MB，无任何依赖，按下热键瞬间激活。

## 运行要求

- Windows 10 / 11 (64位)
- 必须以管理员权限运行

## 安装

直接运行 Nexus.exe 即可。程序启动后没有主窗口，只在系统托盘显示一个图标。

## 使用方法

1. 确保目标窗口在屏幕上可见。
2. 按下全局热键（默认 Ctrl+Alt+K）。
3. 屏幕上所有可点击的元素会被彩色边框框住，并在元素上方显示双字母标签，如 AA、AB、AC。
4. 快速按下对应的双字母组合（例如 AA），即可触发鼠标点击。
5. 按 Tab 键切换点击模式：左键 -> 右键 -> 中键 -> 左键双击 -> 左键。
6. 按 Esc 键退出激活模式，恢复键盘正常使用。

按下第一个字母后，所有不匹配的标签会自动变暗，只剩下匹配项保持高亮，方便快速定位。

## 配置文件

程序首次运行时会自动在以下位置创建配置文件：

    %APPDATA%\Nexus\config.ini

你可以手动修改该文件来调整行为，修改后需重启程序生效。

可配置项：

- Hotkey：全局热键字符串，如 Ctrl+Alt+K
- TimeoutMs：双字母组合的超时时间（毫秒）
- FontSize：标签字体大小
- BgAlpha：标签背景透明度（0-255）
- CornerRadius：标签圆角半径
- TagPosition：标签显示位置，top（元素上方）或 bottom（元素下方）
- Modes：Tab 切换的鼠标模式顺序

## 构建

需要 Visual Studio 2022、Windows SDK 和 C++20 支持。用 VS 打开 `Nexus.slnx`，选择 Release | x64，直接编译即可。

## 下载

前往 [Releases](https://github.com/LLLichet/Nexus/releases) 页面下载最新版本。

## 退出程序

右键点击系统托盘图标，选择 Exit 即可退出。
